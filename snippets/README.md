# Code Snippets

Selected frontend and integration patterns from the 3D service landing page prototype.

| File | Purpose |
|---|---|
| `api-contact-graph.ts` | Contact API route that sends email through Microsoft Graph using application credentials. |
| `reveal-on-scroll.tsx` | Framer Motion reveal wrapper triggered by viewport visibility. |
| `typewriter-on-view.tsx` | Typewriter animation that starts only after the heading enters the viewport. |
| `structured-data-jsonld.tsx` | JSON-LD LocalBusiness structured data component. |
| `nextjs-seo-metadata.ts` | App Router sitemap, robots, and metadata patterns. |

The snippets use placeholders for tenant IDs, mailbox addresses, contact details, and business-specific content. Real credentials and production values are not included.
